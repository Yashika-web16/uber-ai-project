# 🚖 AI-Powered Uber/Ola Ride Optimization System

An end-to-end AI-powered ride optimization platform inspired by Uber and Ola.  
This project combines Machine Learning, Route Optimization, Driver Allocation, and Interactive Visualization into a modern dashboard application.

# 🌟 Features

## ✅ Fare Prediction using Machine Learning
- Built using XGBoost Regressor
- Predicts ride fares based on:
  - Pickup & Drop coordinates
  - Passenger count
  - Date & Time features

---

## ✅ Route Optimization
- Uses NetworkX graph algorithms
- Finds shortest and most efficient paths
- Traffic-aware routing simulation

---

## ✅ Driver Allocation System
- Assigns nearest driver intelligently
- Reduces passenger wait time
- Simulates ride-sharing dispatch systems

---

## ✅ Interactive Dashboard
- Built with Streamlit
- Modern dark-themed UI
- Interactive analytics and visualizations

---

## ✅ Live GPS Maps
- Pickup & drop visualization
- Route rendering using Folium
- Real-time interactive maps
